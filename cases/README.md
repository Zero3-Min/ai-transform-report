## 使用说明

### 按读者关注点分类

- **想了解“AI转型能做多大”**：看案例01（新华智云）——从内部工具到对外产品化的完整路径。
- **想了解“效率提升是否会牺牲内容质量”**：看案例02（界面财联社）——三审机制下的效率提升范式。
- **想找“和我规模最像”的参考**：看案例03（瑞安市融媒体中心）——县级融媒体的真实落地过程，包括其局限性。
- **预算有限、想先试水再决定是否自建**：看案例04（天下秀）——采购第三方工具的路径与风险提示。
- **想知道“AI是不是万能药”**：看案例05（遥望科技）——头部机构的反向教训。

### 引用规范提醒

按 CLAUDE.md 数据可靠性规则，各案例文件内所有量化数字均已标注来源；凡标注“该数字未经证实”的内容，报告正文引用时**不得**省略这一限定语，或应改用案例中已核实的替代数据点。

### 案例库的局限性说明

1. 本次调研聚焦“有公开报道”的案例，未能覆盖大量未公开披露 AI 应用细节的中小传媒公司（这类公司恰恰是本报告最主要的目标客户群体），案例库的代表性有一定局限。
2. 案例 03、04 因规模/性质更贴近目标读者，但公开信息深度不及案例 01、02 两个头部标杆，反映的正是“腰部以下机构效果数据难以获取”这一行业现实，也从侧面印证了本报告“效果度量与ROI模型”（第10章）为何需要客户自行建立基线并追踪，而非直接套用同业数据。
3. 建议实施过程中持续收集客户自身的一手案例数据，逐步替换或补充本案例库中依赖第三方转载的内容。
