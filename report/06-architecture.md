# 技术架构与选型

### 6.1 分层架构

```
应用层   问答界面 / 写作助手 / 预审插件（飞书、企微、浏览器内嵌）
编排层   Dify / FastGPT（工作流、Agent、提示词版本管理）
知识层   向量检索 + 关键词混合检索；文档解析（RAGFlow 可选，见6.2）
模型层   模型路由中间层：国产合规大模型（对外内容）+ 可替换设计
数据层   私有化存储 / 国内云；权限、日志、脱敏
```

### 6.2 选型建议

- **平台（分预算敏感型/标准型两档）**：四个主流开源平台并非单纯的"功能多少"之分，而是各有侧重——Dify（应用开发全能，工作流与 Agent 能力最强）、MaxKB（开箱即用最简单，部署门槛最低）、FastGPT（知识库优先，社区版即具备细粒度权限）、RAGFlow（文档理解最强，但硬件门槛也最高）。**预算敏感型**（团队规模较小、优先低成本起步）推荐 MaxKB——部署门槛四者最低，且提供"问答→工作流→Agent"的渐进式升级路径，与本报告第 4 章三层递进蓝图吻合，不需要中途换平台。**标准型**（预算相对充足、目标是完整跑通阶段 1–2）推荐 Dify 为主平台，社区活跃度断层领先，生态插件丰富，出问题更容易找到解决方案；若客户档案库权限隔离要求严格，可用 FastGPT 替代 Dify 作为主平台（FastGPT 社区版即具备位运算级细粒度 RBAC，优于 Dify/MaxKB 社区版）[来源: FastGPT 官方文档, 《团队&成员组&权限》, https://doc.fastgpt.cn/zh-CN/guide/workspace/team/team_roles_permissions]。
- **开源协议提示**：Dify、FastGPT 的开源协议均规定未经书面授权不得运营与其类似的多租户 SaaS 服务，但这一限制针对"把开源版本包装成对外销售的 SaaS 产品"，客户作为内部工具单租户使用不受影响；MaxKB 为 GPL v3，其"传染性"义务仅在对外分发修改后的代码时触发，内部部署使用同样不受影响；RAGFlow 为标准 Apache 2.0，限制最少。**唯一例外**：若客户未来推进第 4 章 P3 级场景"AI 内容中台对外服务"（即把系统包装后卖给自己的客户），需提前评估购买 Dify 企业版授权或改用 RAGFlow。
- **知识层文档解析（RAGFlow 何时引入）**：RAGFlow 的 DeepDoc 深度文档理解（OCR+表格识别+版面分析）显著强于 Dify/MaxKB 的基础文本抽取，但官方建议至少 32GB 内存，部署门槛四者最高。**仅当客户档案库或爆款拆解库中存在大量复杂 PDF、扫描件或含复杂表格的文档时，才建议引入 RAGFlow 做知识层增强；如客户资料以规范化文本为主，Dify/MaxKB 自带的基础解析已经够用**，避免不必要的硬件投入。
- **模型策略**：对外发布内容优先选用已备案的国产大模型（合规稳妥）；内部提效场景可混用。中间层保证模型可替换，避免锁定。浙江瑞安市融媒体中心 2025 年 2 月直接接入 DeepSeek-R1 API（非自建模型）上线"瑞融AI"，验证了本报告"接入已备案国产大模型、不自研底层模型"这一路线在小规模机构中同样可行。详见 [cases/case-03-瑞安市融媒体中心.md](../cases/case-03-瑞安市融媒体中心.md)。
- **部署（分平台成本估算）**：≤50 人团队可云端托管起步，但月成本因平台而异——MaxKB/Dify 最低配置（2核4GB）云主机月成本约 100–200 元级；FastGPT 推荐配置（4核16GB）约 400–600 元级；RAGFlow 因内存需求高（32GB起）月成本可能超过 800 元，是 MaxKB 方案的 4–6 倍（**注**：以上为基于通用云主机定价惯例的数量级估算，非平台官方发布数据，正式报价前建议按当前云厂商挂牌价复核）；涉及客户机密档案时上私有化（单台 GPU 服务器或纯 API 模式 + 私有数据库均可）。
- **权限**：客户档案库按商务线隔离；合规红线库全员只读；写权限仅责任人。若选用 Dify/MaxKB 作为主平台，社区版权限颗粒度较粗，细粒度角色拆分（如创建与复制权限分离）需付费版本；若权限隔离是硬性要求，优先考虑 FastGPT 或在预算中预留平台付费升级空间。

> **案例参考**：并非所有预算敏感型客户都需要自建知识库平台。上市公司天下秀推出的 AIGC 营销工具"灵感岛"，面向中小 MCN 与门店提供开箱即用的内容生成、数字人克隆等能力，已为超 2 万家门店完成数字人克隆。对完全不具备自建能力的公司，采购成熟第三方工具可作为比自建更快见效的过渡方案；但需注意灵感岛类创新业务收入占比目前不足 5%，采购前应实测效果而非仅凭平台品牌决策。详见 [cases/case-04-天下秀.md](../cases/case-04-天下秀.md)。
