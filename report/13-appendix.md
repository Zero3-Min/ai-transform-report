## 附录 B：MVP 验收指标表

> 下表达标线为**框架性估算**，基于同类项目经验设定的验收基准，非行业统一强制标准；建议在阶段 0 与客户共同确认后写入《试点目标与验收指标确认书》，必要时按客户具体场景（如知识库规模、内容复杂度）上下浮动。

| 指标 | 验收方式 | 达标线 |
|---|---|---|
| 回答准确率 | 50 题人工抽检双人盲评 | ≥85% |
| 引用溯源率 | 抽检回答中带有效文档引用比例 | ≥95% |
| 无中生有率 | 库外问题回答“未收录”的比例 | 100% |
| 检索响应 | 平均首字响应时间 | <10 秒 |
| 周活跃率 | 试点部门周活跃用户占比 | ≥60% |

### 验收方法的可仲裁性（防止"准确率≥85%由谁说了算"的争议）

验收指标若不可仲裁，则形同虚设。为使"准确率 ≥85%"等指标可被客观判定、争议可解，本方案约定以下验收方法，并写入合同附件：

- **固定评测集**：阶段 0 由甲乙双方共同确认一套评测题集（如 50 题，覆盖高频业务场景与已知红线），封存作为验收基准，验收时不得临时更换题目，避免"换题即变分"。
- **双盲评分**：由甲乙双方各派至少 1 名评审，对同一批回答独立双盲打分（评审不知晓对方评分与内容生成方），减少主观偏差。
- **判定规则**：每题按预先定义的对错标准（如"事实无误且引用有效"记为通过）计分，达标线（如准确率 ≥85%）即该批次通过题数占比。判定标准随评测集一并在阶段 0 固化。
- **争议复评机制**：双方评分分歧的题目，先由双方复评协商；仍无法达成一致的，可约定引入第三方评审或行业专家复评，以复评结果为准。
- **留痕**：评测集、每题评分、争议处理过程全程留档，作为验收结论的可追溯依据。

通过上述设计，"准确率 ≥85%"从一个模糊承诺变为**由固定题集、双盲评分、明确判定规则和争议复评机制共同支撑的可仲裁标准**。
