# 附录 A：诊断访谈问题清单（节选）

**对老板**：过去一年因内容质量/合规损失过什么？如果编辑产能翻倍，多出来的产能想用在哪？愿意为转型每周投入多少自己的时间？
**对主编**：新人从入职到独立出稿要多久、卡在哪？返工最常见的三个原因？公司文风如果用三个词概括是什么、写在哪里？
**对运营**：各平台规则变动你们如何获知与同步？过去半年被限流/下架几次、原因？
**对商务**：客户资料现在存在哪、谁能看？编辑拿错客户口径的事故发生过吗？

## 附录 B：MVP 验收指标表

> 下表达标线为**框架性估算**，基于同类项目经验设定的验收基准，非行业统一强制标准；建议在阶段 0 与客户共同确认后写入《试点目标与验收指标确认书》，必要时按客户具体场景（如知识库规模、内容复杂度）上下浮动。

| 指标 | 验收方式 | 达标线 |
|---|---|---|
| 回答准确率 | 50 题人工抽检双人盲评 | ≥85% |
| 引用溯源率 | 抽检回答中带有效文档引用比例 | ≥95% |
| 无中生有率 | 库外问题回答“未收录”的比例 | 100% |
| 检索响应 | 平均首字响应时间 | <10 秒 |
| 周活跃率 | 试点部门周活跃用户占比 | ≥60% |
