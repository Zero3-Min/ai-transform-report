# 组织与变革管理

- **一把手工程**：老板/主编亲自担任转型负责人，参与阶段 0 目标设定与每月复盘。员工参与度与转型成效强相关——直接参与转型的员工比例是成败的先导指标。
- **共建而非空降**：选 2–3 名主力编辑做"种子用户"，风格指南由他们参与定版——人只会捍卫自己参与制定的规则。
- **角色重定义**：编辑 KPI 从"篇数"转向"策划质量 + 终审质量 + 人均管理内容量"；新增兼职角色"知识管理员"（维护知识库）与"合规员"。
- **叙事管理**：对内统一口径——AI 接走的是初稿苦力活，编辑升级为创意总监。第一个月用"省下的时间"数据做全员通报，固化正反馈。新华社研究院 2024 年面向全球 53 国媒体机构的调研显示，67.6% 的受访机构已明确感知到 AI 带来的变革，其中"生产流程重塑、编辑记者工作方式转变"是最突出的变化之一[来源: 新华社研究院（国家高端智库）, 《人工智能时代 新闻媒体的责任与使命》, 2024年10月14日, https://www.news.cn/politics/20241014/fe72c14c8d1e4e509d68d0ff04455725/c.html]——角色转型是行业普遍趋势而非个别公司的孤立选择，这一点本身就能降低员工的抵触心理。
