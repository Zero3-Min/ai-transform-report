# 合规与风控体系

这是传媒方案区别于其他行业的核心章节，也是咨询服务的差异化卖点。

1. **AIGC 标识合规**：依《人工智能生成合成内容标识办法》（国家网信办等四部门联合发布，2025年9月1日起施行）[来源: 中央网络安全和信息化委员会办公室, 《关于印发〈人工智能生成合成内容标识办法〉的通知》, 2025年3月14日, https://www.cac.gov.cn/2025-03/14/c_1743654684782215.htm]，AI 参与生成的对外内容执行显式标识（按各平台入口申报，第四条）+ 隐式标识留痕；公司作为内容发布者（办法所称"用户"）在发布时须主动声明并使用标识功能，不得删除、篡改、伪造、隐匿标识（第十条）；发布检查清单固化为流水线最后一环。
2. **广告合规**：商务内容强制"广告/赞助"标注；极限词、医疗功效、投资回报承诺列入红线库绝对禁区，预审 Agent 逐条拦截。
3. **版权风控**：AI 输出禁止直接引用长段第三方内容；素材库登记授权状态；数据引用必须可查来源。
4. **审核留痕**：谁生成、谁修改、谁终审、何时发布全程日志，争议时可举证——这同时是对客户（品牌方）的服务承诺。
5. **账号安全**：知识库回答"不编造"原则（无相关知识时明确说"未收录"），从机制上杜绝 AI 幻觉流入发布内容。
