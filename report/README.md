## 一致性检查清单

在进行项目推进或客户沟通时，建议使用本索引中的**关键数字清单**进行口径一致性检查，确保：
- 人均产能提升目标一致（40%）
- 成本投入估算范围准确（轻量1–2万 → 进阶12万+，与第6章分平台部署成本估算属不同费用类别，不可混淆）
- 回收周期预期合理（<2个月）
- 风险认知完整（7大核心风险已识别，含2026年7月新增"技术投入脱离核心业务健康度"）
