# 实施计划（分阶段 WBS）

> 以下各阶段的周数/月数为**框架性估算**，基于同类项目经验制定，实际排期需在阶段 0 诊断后按客户实况（团队规模、知识资产现状、配合度）校准，并写入《试点目标与验收指标确认书》。

### 阶段 0：诊断与共识（第 1–2 周）
- W1：高管访谈（2h×3 人）、员工问卷、知识资产盘点（附录 A 清单）
- W2：成熟度评分、场景清单评审会、签署《试点目标与验收指标确认书》
- 交付物：诊断报告、场景清单、实施计划书

### 阶段 1：知识库 MVP（第 3–8 周）
- W3–4：知识治理冲刺——集中、去重、定版、编号（工作量占比约 60%，客户须投入主编+运营各 0.5 人力）
- W5：平台部署、文档接入、检索调优
- W6：权限配置、种子用户内测、坏例修复
- W7：全员培训（2h）+ 上线
- W8：首次效果复盘，对照附录 B 验收
- 交付物：五大知识库 V1、使用手册、管理员培训

### 阶段 2：创作流水线（第 3–5 个月）
- M3：初稿 Agent（含风格自检输出）灰度上线；预审 Agent 规则库建设
- M4：流水线全流程贯通，发布检查清单固化；周更机制运转验证
- M5：数据复盘（工时、返工率、合规拦截数），流程定版
- 交付物：创作流水线系统、SOP 文档、月度运营报告模板

### 阶段 3：深化与重塑（第 6 个月起，可选续约）
- 选题助手、评论回复、数据报告等 P2/P3 场景滚动上线
- 探索"AI 内容中台"对客户品牌方的产品化输出

> **案例参考**：这一探索方向并非空谈。新华智云正是从内部 AI 工具起步，逐步产品化为对外服务全国 3000+ 媒体机构的"媒体大脑"平台，是"AI 内容中台产品化"路径在头部机构的完整实现。10–200 人公司在阶段 3 可参考其"先内部验证、再对外输出"的节奏，而非一开始就定位为对外产品。详见 [cases/case-01-新华智云.md](../cases/case-01-新华智云.md)。
